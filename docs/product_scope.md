# 产品范围：v0.2.0 智能背词主流程与动态规划闭环版

## 新定位

v0.2.0 不再只是数量规划工具，而是一个本地离线的“真正可背词的软件 + 动态计划中枢 + AI 规划接口”。

核心目标：

- 用户能够学习每日分配的具体单词。
- 用户能够复习具体单词并记录掌握状态。
- 系统严格区分词库供给缺口和用户学习欠缺。
- 动态计划由稳定、可测试的本地逻辑执行。
- AI 只负责目标解析、词书类别建议和解释，不直接绕过本地调度规则。

## 已实现范围

- 结构化目标设置：目标类型、截止日期、目标词量、每日新词上限、每日复习上限、休息日、缓冲日比例、计划类型和计划时区。
- 自然语言目标入口：本地 mock AI 输出结构化建议，用户确认后才填入目标表单。
- 词表导入：CSV / JSON 导入、标准化去重、来源词书保留。
- 新词学习：每日任务绑定具体 `WordItem`，支持“已学习 / 已掌握 / 暂时跳过 / 今日未完成”。
- 今日复习：到期复习绑定具体单词，支持“不认识 / 模糊 / 认识 / 很熟悉 / 未完成”。
- 动态重排：学习欠缺、逾期复习、库存缺口和容量限制共同参与后续排期。
- 计划解释：覆盖状态、不可行原因、调整日志。
- 数据管理：v0.2.0 完整备份；v0.1.0 旧备份迁移为历史数量记录。

## 页面结构

- 总览：目标词量、可供给词量、库存缺口、完成量、待补学、逾期复习和最近调整。
- 目标设置：结构化目标、自然语言目标、AI mock 建议、目标变化预览入口。
- 词书管理：已导入词书真实去重词数、推荐但未导入词书、库存缺口、导入功能。
- 今日学习：新词学习列表和今日复习列表。
- 计划：长期覆盖、月度计划、周度计划、每日任务数量和调整日志。
- 统计：掌握、复习中、未开始、库存缺口、学习欠缺、逾期复习和旧版历史数量记录。
- 数据管理：导出、导入、版本识别和清空数据。

## 本轮不实现

- 真实 AI API。
- 发音播放、例句生成、拼写训练、听力训练。
- 登录、云同步、多端同步。
- 自动抓取商业词书或来源不明词库。
- v0.2.0 正式 tag / Release。该步骤等待验收合并后执行。
