# v0.4.0 - 动态进度恢复与计划视图重构版

## 版本定位

v0.4.0 聚焦期限约束下的动态进度恢复。用户漏学或少背后，系统保留具体未完成单词，在不改变总目标和截止日期的前提下，把待补学任务尽量平滑地安排到后续有效学习日，并在无法按期完成时给出明确原因。

## 主要变化

- 重构动态恢复算法，未来新词不再简单按每日上限从前往后填满。
- 新增总负荷模型：`新词 + 复习 * 0.6`。
- 待补学任务优先放入负荷较低日期，避开复习高峰。
- 上限不足时显示剩余新词、剩余学习日、最低日均新词和当前上限。
- 新增今日任务卡片、周计划卡片、月历 / 负荷热力图、阶段时间轴和详细明细。
- 统一中文状态文案，避免界面裸露内部枚举。
- 没有真实目标词表时，六级自然语言目标不再生成 900 词这类无依据正式建议。
- 备份导出版本更新为 `v0.4.0`，继续兼容旧版本导入。

## 不包含

- 不扩充正式词书库。
- 不接入真实 AI API。
- 不新增发音、例句、拼写训练。
- 不新增登录、云同步或应用商店发布。
