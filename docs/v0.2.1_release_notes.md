# v0.2.1 - 技术基线归档说明

## 版本定位

本文件是 v0.2.1 开发完成时准备的发布说明归档。v0.2.1 修复了 v0.2.0 的逐词操作与当日结算一致性问题，并成为 v0.3.0 的技术基线。

当前实际状态：Git 中存在 `v0.2.1` tag，但 GitHub Releases 页面没有 v0.2.1 Release。本轮不再单独发布 v0.2.1；其有效能力由 v0.3.0 完整承接。

## 新增功能

- 支持绑定具体单词的每日新词任务。
- 支持具体单词级复习任务与复习结果记录。
- 支持词库供给缺口与用户学习欠缺任务的严格区分。
- 支持导入新词书后补足目标缺口并重新排期。
- 支持自然语言目标输入入口与 AI 规划 mock 结构。
- 支持 v0.2.1 完整数据备份与恢复。

## 核心修复

- 修复处理单个今日新词后，当天其余未处理任务被提前计入待补学的问题。
- 修复处理单个今日复习词后，当天其余未处理任务被提前计入逾期的问题。
- 新增“结束今日学习并安排未完成任务”的明确结算流程。
- 新增跨日自动结算与重复结算幂等性处理。
- 保留 v0.2.0 备份读取兼容。
- 保留 v0.1.0 legacy 数量记录迁移逻辑，不伪造历史具体单词记录。

## 本版暂未包含

- 真实 AI API 接入。
- 发音与例句。
- 拼写训练。
- 云同步与账户系统。
- 词书优先级推荐算法重构。
