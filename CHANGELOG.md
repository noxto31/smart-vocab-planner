# Changelog

## v0.2.0 - 智能背词主流程与动态规划闭环版

### 新增

- 具体单词级新词任务。
- 具体单词级复习任务与掌握状态记录。
- 词库供给缺口与学习欠缺任务的严格区分。
- 新词学习页与今日复习流程。
- 自然语言目标输入入口与 AI 规划服务 mock 接口。
- 新词书导入后补足缺口并动态重排。
- Dexie v2 数据表、v0.2.0 完整备份和 v0.1.0 备份兼容迁移。

### 修复

- 修复目标数量与真实词库脱节的问题。
- 修复复习任务只有数量、没有具体单词关联的问题。
- 修复历史补录或重复提交可能造成的数据一致性问题。
- 修复固定使用 Asia/Shanghai 时区造成的日期边界错误。

### 暂未包含

- 真实 AI API 接入。
- 发音、例句、拼写训练。
- 登录与云同步。
- 正式商业词书数据抓取。

## v0.1.0 - 离线数量计划 MVP 基线版

### 已实现

- 目标设置与基础词表导入。
- 数量级计划生成。
- 基础每日完成记录与动态重排。
- 可行性判断。
- 备份恢复与基础测试。

### 当前限制

- 未实现具体单词级学习任务。
- 未实现真实复习闭环。
- 词库不足与用户未完成任务尚未严格区分。
